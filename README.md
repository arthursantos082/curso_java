# Curso Java

Repositório dedicado aos estudos e práticas desenvolvidas ao longo do curso Java COMPLETO Programação Orientada a Objetos + Projetos, do professor Nelio Alves. O objetivo é consolidar os fundamentos da linguagem e evoluir gradualmente para conceitos mais avançados de programação.

## Sobre o projeto

Este repositório reúne conteúdos, anotações e exercícios práticos abordando desde lógica de programação até Programação Orientada a Objetos (POO), utilizando a linguagem Java.

Os códigos aqui presentes fazem parte do processo de aprendizado e servem como apoio para fixação dos conceitos estudados.

## Conteúdo abordado

- Introdução à linguagem Java  
- Lógica de programação  
- Estruturas sequenciais  
- Estruturas condicionais  
- Estruturas de repetição  
- Manipulação de dados e variáveis  
- Arrays e listas  
- Conceitos de Programação Orientada a Objetos:
  - Classes e objetos  
  - Encapsulamento  
  - Construtores  
  - Herança  
  - Polimorfismo  

## Estrutura do repositório

O repositório está organizado em pastas que representam aulas, módulos ou tópicos do curso. Cada diretório contém exemplos práticos e exercícios relacionados ao conteúdo estudado.

## Tecnologias utilizadas

- Java  
- IDE (Eclipse, IntelliJ ou similar)  
- Git e GitHub  

## Objetivo

O principal objetivo deste repositório é:

- Praticar a linguagem Java  
- Consolidar fundamentos de programação  
- Servir como material de consulta futura  
- Demonstrar evolução no aprendizado  

## Como executar os projetos

1. Clone o repositório:
```
git clone https://github.com/arthursantos082/curso_java.git
```

2. Abra o projeto em sua IDE de preferência  

3. Compile e execute os arquivos `.java` conforme necessário
